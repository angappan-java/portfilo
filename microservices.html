PROJECT 1

Project Name :  Store Management Web Based Application

Platform :  Microservice

Teck Stack :  Java, Spring Boot, Spring Cloud(API Gateway), Postman, RESTful APIs, Spring Data JPA, Hibernate, MSSQL, Thymeleaf, HTML, CSS, Javascript, Bootstrap

Project Desciption :  Developed a small-scale Microservices-based Product and Order Management System using Spring Boot and Spring Cloud. The architecture consists of four independent services Product Inventory, Order Inventory, API Gateway (with YAML routing configuration), and Store Management frontend each running on separate port. Implemented RESTful APIs with Repository-Service-Controller layer in Product and Order Service, Integrated Thymeleaf based on frontend using RestTemplate via API Gateway for viewing and managing product and order details and thoroughly tested all endpoints using Postman.
